# 🎓 ClubConnect

Welcome to **ClubConnect**! 

This is a modern, responsive web application designed to help university students discover, explore, and register for various campus clubs and events. Whether you're into programming, arts, sports, or literature, ClubConnect makes it super easy to find where you belong.

## ✨ Features

- **Explore Clubs**: Browse through all the active clubs on campus, beautifully categorized (Technical, Cultural, Sports, Literature).
- **Club Details**: Get all the information you need before joining, including timings, durations, current member counts, and the exact on-campus venue.
- **Event Registration**: Stay updated with upcoming campus events and sign up directly through an intuitive registration form.
- **Dark/Light Mode**: Fully customized theming because staring at a bright screen at 2 AM is no fun!
- **Favorites**: Follow the clubs you love to keep track of them in your personalized profile.

## 🛠️ Built With

This project was built entirely from scratch focusing on modern frontend aesthetics:
- **React.js** (via Vite for lightning-fast builds)
- **Vanilla CSS** with advanced techniques like:
  - *Glassmorphism* for that premium, frosted-glass look.
  - *CSS Variables* for seamless theme switching.
  - *CSS Grid/Flexbox* for a fully responsive, mobile-friendly design.
  - *Micro-interactions* and hover animations for a dynamic user experience.

## 🚀 Getting Started

If you want to run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pallavi14bhardwaj/club-connect.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd club-connect
   ```
3. **Install the dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173` in your browser to see the app live!
